DOWNLOADER
===========

A simple java program to download content of given URLs to the given location.It accepts a list of URLs and a file location.Inputs are read from config.properties.Directory will be created in case it does not exist as per input provided in location field.


HOW TO RUN IT
=============
This program can be executed running Application.java file.


PREREQUISTE
===========
A comma separated list of URL is expected and a single location is expected.

Example : 
url=URL1,URL2,URL3
location=C:\\Downloader


Built With
==========
Maven
